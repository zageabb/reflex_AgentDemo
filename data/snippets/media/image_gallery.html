Snippet with image support

Use snippets to pair the narrated steps with visuals. Upload PNG, JPG, SVG, or GIF assets through the admin dashboard and reference them with the /snippet/<path> URL so they render inline.

  1. Upload your image to instance/uploads/ or store it under data/snippets/.
  2. Reference the file from your scenario step: "snippet": "media/image_gallery.html".
  3. Embed the asset using an absolute snippet path, e.g. /snippet/media/snippet-preview.svg.
  4. Add descriptive alt text so presenters know what the audience sees.
